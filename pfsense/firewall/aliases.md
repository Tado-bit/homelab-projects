# pfSense Firewall Aliases

## MGMT_LAN
- Network: 192.168.1.0/24
- Purpose: pfSense management access only

## SERVER_LAN_NET
- Network: 192.168.2.0/24
- Purpose: Server isolation network

## SECURITY_LAN_NET
- Network: 192.168.3.0/24
- Purpose: Security testing and monitoring network

## INTERNAL_NETS
- 192.168.1.0/24
- 192.168.2.0/24
- 192.168.3.0/24
- Purpose: All internal RFC1918 LANs
